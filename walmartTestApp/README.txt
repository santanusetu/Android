/** Building the MovieAppTest project in your workspace **/
===========================================================


To build this project: 

* Import MovieAppTest, actionbarsherlock and library in your workspace
  
  library => implements https://github.com/jfeinstein10/SlidingMenu library
  Add actionbarsherlock as a library in the library project

* Add library as a library to the MovieAppTest project

* Clean the project - build it - run it in your system
