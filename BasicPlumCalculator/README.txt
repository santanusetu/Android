- Basic plum calculator that handles plus, minus and clear operation.
- Supports operations upto 7 digits in the screen
- Supports multiple operation sequence
- Handles erronous operator entry by taking account of the last pressed operator only

Building the Application: 
The application was built using Eclipse with ADT.
Import the project as a new android project in your workspace and build it.
Make sure the appcompat v7 support library is there in your workspace.
